# test-3DGame-threeJS 
This is my first 3D game with ThreeJS. 
You have a hero - Stickman, he runs and catchs the brains. Your goal - to take a 200 points. After that Stikman will stop and winner-dancing. Else he will stop and die. 
Control: key "A": left,
key "D": right,
kew "S": middle.
